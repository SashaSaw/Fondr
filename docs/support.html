Linking with your partner

Fondr is built for two. To use the app, you need to pair with your partner using an invite code. One person creates the code, the other enters it.

1

Sign in

Open Fondr and sign in with your Apple account or email. Complete the short onboarding to set your display name.

Screenshot: Sign-in screen
2

Create an invite code

On the pairing screen, tap Create Invite Code. A unique 6-character code will be generated for you. Share this code with your partner via text, email, or any way you like.

Screenshot: Invite code screen with code displayed
3

Your partner enters the code

Your partner opens Fondr, signs in, and taps I have a code. They enter the 6-character invite code you shared and tap Join Partner.

Screenshot: Join partner screen with code entry
4

You're paired!

Once your partner enters the code, you're instantly connected. Both of you will see the main app with all shared features unlocked.

Screenshot: Couples home page after pairing
Tip: Each invite code can only be used once. If the code expires or doesn't work, simply go back and create a new one.

Shared lists

Create shared lists for anything — date ideas, restaurants to try, movies to watch, or places to visit. Both partners can add, edit, and check off items in real time.

Screenshot: Shared lists view

Swipe to decide

Can't agree on what to do? Start a swipe session from any list. Both partners swipe through the items independently — when you both swipe right on the same item, it's a match. No more endless "you pick" conversations.

Screenshot: Swipe session

Shared calendar

Mark your available days, see when your partner is free, and find overlapping time for your next date. Propose events and your partner can accept or suggest changes.

Screenshot: Calendar with shared availability

Account & settings

From the Settings tab you can:

  • Update your display name and profile photo
  • Set your anniversary date
  • Leave a partnership (this removes all shared data)
  • Delete your account
